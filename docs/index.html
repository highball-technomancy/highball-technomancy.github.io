Highball Technomany presents: Permafrost

Permafrost (working name) is a next-generation cryptocurrency grid trading bot, designed for traders who demand automation, transparency, and innovative features.

With an AI-assisted strategy core and a modern, real-time dashboard, Permafrost empowers users to run complex, multi-market grid trading with deep observability and risk control.

Core Features

  • Grid Trading Automation – Buy low, sell high at predefined levels across multiple pairs, 24/7.
  • AI-Powered Analysis – Get smart market insights and strategy recommendations informed by advanced machine learning.
  • Real-Time Monitoring – Lightning-fast updates with full observability through Grafana, InfluxDB, and live charts.
  • Risk-Free Paper Trading – Sandbox mode lets you develop and test strategies with zero capital at risk.
  • Advanced API & Modular Design – FastAPI backend, React UI, robust API, and Dockerized deployments for easy extension.
  • Multi-Exchange Support – Compatible with industry-leading crypto exchanges (e.g. Coinbase, OKX).
  • Open Source – MIT-licensed and transparent. Visit GitHub for code and contributions.
View Project on GitHub

Technology Stack

  • Backend: Python 3.11, FastAPI, SQLAlchemy, Redis, InfluxDB, CCXT, Pandas/NumPy/Scikit-learn
  • Frontend: React (TypeScript), Lightweight-Charts, WebSockets
  • Infrastructure: Docker Compose, PostgreSQL 16, Grafana
